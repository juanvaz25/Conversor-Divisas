# convertidor-divisas

Aplicacion realizada en VUE 3 para convertir divisas

## Como utilizar

Clonar repositorio -> abrir consola en la raiz del repositorio
Seguir con el Setup

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
